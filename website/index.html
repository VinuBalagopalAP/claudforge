CLI Reference

claudforge — batch deployment
Copy
Usage: claudforge [command] [PATH] [options]

Global Options

--limit N

Strictly attempt N brand-new uploads (highly recommended for stabilizing daily quotas).

--force, -f

Ignore local history; re-verify every skill on the Cloud (bypasses cache).

--profile NAME

Use a persistent Chrome profile (preserves active sessions and cookies).

--headless

Run the deployment engine in background mode without a visible browser UI.

--connect [PORT]

Attach to a pre-opened Chrome instance (skips redundant login/2FA).

--keep-zips

Persist temporary skill bundles in _zips folder for manual auditing.
